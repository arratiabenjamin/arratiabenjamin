  ### 🔔 [Claude Notch](https://github.com/arratiabenjamin/claude-notch)
  
  App **nativa de macOS** escrita en **Swift 6 + SwiftUI + AppKit** que convierte el notch de tu Mac en un Dynamic Island para **Claude Code**.
  Mostrá en vivo cuántas sesiones de Claude tenés corriendo en paralelo (Velion, Workly, side projects…), su estado (`running` / `idle` / `ended`), y todo con material de cristal real (`NSVisualEffectView` + `.hudWindow`) sobre un panel flotante que respeta el silueteado del notch.                                                                                                                                                                                                     
                                                         
  **Highlights técnicos:**                                                                                                                                                                                                                          
  - 🪟  `NSPanel` flotante con `liquid glass` + `NotchPillShape` que sigue la curvatura del hardware
  - ⚡ Reactividad instantánea vía **FSEvents** sobre `~/.claude/active-sessions.json` (zero polling)
  - 🧱 **Clean Architecture** lite (App / Domain / Infrastructure / Presentation)
  - 🖱️  Hover-out collapse con global mouse monitor + bridge rect entre notch y panel
  - 🛠️  **XcodeGen** como source of truth (`project.yml` versionado, `.xcodeproj` regenerable)
  
  Hermano del proyecto [`claude-code-notifier`](https://github.com/arratiabenjamin/claude-code-notifier), que es el que escribe el state-file desde los hooks de Claude Code. Boundary limpio: claude-notch es **readonly consumer**.
